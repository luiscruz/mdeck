# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Build Command

Convert the Markdown slides to PDF using `maked`:

```sh
maked slides.md
```

This invokes Pandoc with the Beamer backend and LuaLaTeX engine, reading build options from the file's YAML frontmatter.

## Architecture

This is a Pandoc + Beamer (LaTeX) slide deck project.

- `slides.md` — the source presentation in Pandoc Markdown with YAML frontmatter. Each `# Heading` becomes a slide frame. Horizontal rules (`---`) can also delimit frames.
- `beamerthememetropolis.sty` and companion `beamer*metropolis*.sty` files — local copies of the [Metropolis](https://github.com/matze/mtheme) Beamer theme, included so no system-wide installation is required.
- `slides.pdf` — the compiled output; regenerated by the build command above.

### Frontmatter options (slides.md)

| Key | Purpose |
|-----|---------|
| `theme` | Beamer theme (currently `metropolis`) |
| `aspectratio` | Slide aspect ratio (e.g. `169` for 16:9) |
| `mainfont` / `mainfontfallback` | LuaLaTeX font settings |
| `pdf-engine` | Must be `lualatex` for Unicode font support |

### Dependencies

- `maked`
- `pandoc`
- `lualatex` (part of a TeX distribution such as TeX Live or MacTeX)
- Fonts: NotoSans, NotoColorEmoji
